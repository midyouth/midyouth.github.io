3. 电脑环境配置

3.1 系统配置

  • Windows系统：统一使用 Windows Linux子系统 并且配置 Cmder Shell
  • Mac系统：统一使用 系统自带Terminal 并且配置 Iterm2
  • Linux系统：推荐使用 Deepin国内发行版 或 Ubuntu标准版，使用 U盘驱动安装
  • 其他：此外也可以选择 VMware虚拟机操作 或制作 U盘便携系统

3.2 环境配置

默认你完成【1.1系统配置】后当前处于Unix系统的Terminal中进行以下操作
* 编程语言：Python
* 包裹管理：Pip 配置 以及 Brew配置
* 虚拟环境：Virtualenv
可以直接用以下命令行配置并测试

  1. pip install virtualenv;
  2. cd ~;
  3. mkdir MidYouthProject;
  4. virtualenv -p python3 runtime;
  5. source runtime/bin/activate;
  6. python

此时需要确保屏幕出现Python3.6/3.7版本的交互模式
