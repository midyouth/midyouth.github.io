1)【向世界问好】 - 用你的命行说出 Hello World!

echo 中文意义为 回音，而其的功能也正如他的名字一样 - 就像是在发出回音。

在命令行中我们可以使用echo后加上一串文字，而不论我们加上任何文字都将像是回音一样被系统再显示在下一行中。

那就让我们一起用echo向世界问好把！

任务：在命令中输入以下内容(忽略以#符号开头的注释)

  1. echo Hello World! 
  2. # 屏幕将返回 Hello World!
  3. echo I'm [yourname]
  4. # 讲[yourname]替换成你的名字拼音或英文名

2)【云上系统里走走转转】

请完成以下示范操作

Linux中用 pwd 命令来查看”当前工作目录“的完整路径，因为pwd全拼音为[print working directory]，意为“打印当前工作目录”。

简单得说，每当你在终端进行操作时，你都会有一个当前工作目录。在不太确定当前位置时，就会使用pwd来判定当前目录在文件系统内的确切位置。例如：

  1. pwd
  2. # 系统将显示你当前所在的文件夹，如/usr/students/xiaoming

与pwd相应的是mkdir, 全拼音为[make directory]，即为建立新文件夹，而mkdir后面的字符则为即将新创建的文件夹的名称，例如

  1. mkdir test_folder
  2. # 新建一个名字为test_folder(测试文件夹)的文件夹

新建完文件夹后，我们可以通过ls，意味[list] - “列出”来查看当前文件夹中所存放的文件和其他文件夹。

  1. ls
  2. # 系统将显示你刚刚所创建的test folder

同时，我们可以使用cd来切换当前所在的文件夹。cd意味[change directory],中文则为“改变文件夹”。

cd后面可以加上一个文件夹的名字，从而进入到所指定的文件夹。也可以加上一个特殊符号.., 通常这个特殊符号都用来表示“上一层文件夹”。最后，如果不在cd后面添加任何文件夹的名称，则将返回到默认主文件夹。
例如

  1. cd ..
  2. pwd
  3. ls
  4. # 切换到上一层目录并查看所在目录，随后查看该目录中存在的信息
  5. cd
  6. pwd
  7. # 切换回该账户默认目录并查看当前所在目录
